RPC (Remote Procedure Call)

A simple RPC implementation.

Example: Ping

Structs PingRequest and PingResponse are defined as examples to demonstrate the usage of the RPC library. They implement the RpcRequest and RpcResponse traits respectively.

This example demonstrates the basic usage of the RPC library by sending a ping request to a set of nodes and waiting for a response.

Start some docker containers with the internal docker network attached. And run the following command to start the example:

cargo run --example ping 172.19.0.2:16

Note that the IP address and port number are the address of the current node.

Or simply run the PowerShell script which handles everything for you:

.\examples\ping.ps1
